# Object-Tracking
Several algorithms and techniques to perform object tracking and detection using python and openCV

## ALGORITHMS USED:
- CSRT
- KCF
- MedianFLow
- TLD
- Mosse
- Boosting 
- MIL
- MeanShift
- CAMShift
- Optical Flow Sparse
- Optical FLow Dense
